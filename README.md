# 校园网络算法可视化平台

全功能的前后端分离项目，实现了最小生成树、最大流、AES加密算法和交互式流量仿真等多种网络算法的可视化演示。

## 🚀 快速开始

### 1. 启动后端
```bash
cd backend
pip install -r requirements.txt
python app.py
```
后端服务: `http://localhost:5000`

### 2. 启动前端
```bash
cd frontend
npm install
npm run dev
```
前端服务: `http://localhost:3000`

## ✨ 功能特性

### 核心算法
- 🌲 **最小生成树 (MST)**: Kruskal & Prim 算法的对比与步骤可视化
- 💧 **最大流 (Max-Flow)**: Edmonds-Karp & Dinic 算法的性能对比与动画演示
- 🔐 **AES加密**: 完整的AES-128加密/解密实现

### 网络功能
- 🌐 **智能网络生成**: 随机生成连通平面网络，支持自定义节点数、造价和容量范围
- 🖼️ **拓扑可视化**: 高质量网络拓扑图生成，支持多种标签模式
- 🎮 **交互式仿真**: 实时流量仿真，支持多路径负载均衡和拥塞检测
- 📊 **流量监控**: 动态链路利用率监控、拥塞分析和负载均衡优化

### 可视化特性
- 🎬 **步骤动画**: 算法执行过程的逐步可视化
- 📈 **实时统计**: 性能指标、流量统计和网络状态监控
- 🎨 **交互式界面**: 现代化UI设计，支持缩放、拖拽等交互操作
- 🔧 **灵活配置**: 支持手动编辑网络拓扑和算法参数

## 📖 使用指南

### 基础使用流程
1. **网络配置**: 在"网络配置"标签页生成或手动创建网络拓扑
2. **应用配置**: 点击"应用到算法"将网络设为所有算法的默认输入
3. **算法执行**: 在各功能标签页运行相应算法并查看结果
4. **结果分析**: 通过可视化图表和动画理解算法执行过程

### 高级功能
- **算法对比**: 同时运行多种算法，对比性能差异
- **参数调优**: 调整网络参数，观察对算法性能的影响
- **流量仿真**: 使用交互式仿真功能进行网络容量规划
- **数据导出**: 支持算法结果和可视化图像的导出

## 🛠 技术栈

### 后端技术
- **框架**: Flask 3.0 + Flask-CORS
- **图算法**: NetworkX + 自实现核心算法
- **可视化**: Matplotlib (服务端图像生成)
- **数学计算**: NumPy
- **加密**: 自实现AES-128算法

### 前端技术
- **框架**: Vue 3 Composition API + Vite
- **可视化**: 原生SVG + 动画效果
- **状态管理**: Vue 3 provide/inject
- **样式**: 现代CSS + 响应式设计

## 📊 算法复杂度对比

| 算法 | 时间复杂度 | 空间复杂度 | 特点 |
|------|-----------|-----------|------|
| Kruskal MST | O(E log E) | O(V) | 边排序 + 并查集 |
| Prim MST | O(E log V) | O(V) | 优先队列 |
| Edmonds-Karp | O(VE²) | O(V²) | BFS增广路径 |
| Dinic | O(V²E) | O(V²) | 层次图 + DFS |
